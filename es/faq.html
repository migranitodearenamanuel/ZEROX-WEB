Preguntas Frecuentes

Detalles técnicos para operadores e ingenieros.

¿Qué necesito para ejecutar ZEROX?

Python 3.9 o superior. Recomendamos un VPS Linux (Ubuntu 22.04) con al menos 2GB de RAM para ejecución headless 24/7. Funciona en Windows/Mac para desarrollo.

¿Es realmente autónomo?

Sí. Una vez configurado el archivo .env y los parámetros en settings.json, ZEROX inicia el bucle principal y solo se detiene ante errores críticos o señal de apagado manual.

¿Cuánto cuesta?

El núcleo es Open Source (MIT License). Eres libre de usarlo, modificarlo y desplegarlo. No cobramos por el software.

¿Puedo contribuir?

Absolutamente. Aceptamos Pull Requests en GitHub que mejoren la estabilidad o añadan nuevos módulos de integración.
