SAFETY FIRST

Safety Protocols

Speed is nothing without control. ZEROX prioritizes capital integrity and operation stability.

Circuit Breakers

If API latency exceeds 200ms or consecutive 5xx errors are detected, the system triggers "Sleep" mode automatically.

Hard Limits

Immutable runtime variables define maximum daily loss. No impulsive human "override" is possible.

Process Isolation

The execution module runs on a separate thread from analysis. A calculation failure does not block the emergency exit.

Immutable Logs

Every decision is written to disk before execution. Complete forensic traceability for post-mortems.
