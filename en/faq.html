Frequently Asked Questions

Technical details for operators and engineers.

What do I need to run ZEROX?

Python 3.9 or higher. We recommend a Linux VPS (Ubuntu 22.04) with at least 2GB RAM for 24/7 headless execution. Works on Windows/Mac for dev.

Is it truly autonomous?

Yes. Once `.env` and `settings.json` are configured, ZEROX starts the main loop and only stops on critical errors or manual kill signals.

How much does it cost?

The core is Open Source (MIT License). You are free to use, modify, and deploy it. We do not charge for the software.

Can I contribute?

Absolutely. We accept GitHub Pull Requests that improve stability or add new integration modules.
