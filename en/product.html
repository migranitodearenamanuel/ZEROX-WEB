CORE MODULES

Modular Intelligence

Each component has a unique and isolated responsibility. Low coupling design for maximum robustness.

Master Mind

mente_maestra.py

The central brain. Processes signals, queries RepoBrain (RAG), and determines operational intent. It plans, but does not execute.

Risk Manager

gestor_riesgo.py

The supreme security layer. Validates every intent against predefined limits. If the risk is high, it blocks execution. Final authority.

Master Operator

operador_maestro.py

The execution arm. Translates validated intents into specific protocol commands for external environments. Manages latency.

Visual Bridge

puente_visual.py

Telemetry interface. Exposes the internal system state in real-time to the frontend without compromising the main loop performance.
